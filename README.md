# Проект react-mesto-api-full
 
## Цель проекта

Написать приложение с возможностью создания и редактирования профиля пользователя,
добавления, просмотра, постановки лайков и удаления фотографий. 
  
## Функционал

Создание базы пользователей, авторизация пользователей, добавление и удаление фотографий, просмотр фотографий, редактирование профиля, валидация.

## Директории

Репозиторий для приложения проекта `Mesto`, включающий фронтенд и бэкенд части приложения со следующими возможностями: авторизации и регистрации пользователей, операции с карточками и пользователями. Бэкенд в директории `backend/`, фронтенд - в `frontend/`.

## Запуск проекта

`npm start` — запускает сервер.
